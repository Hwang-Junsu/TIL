# Selector
- 꾸밀 요소를 선택하는 선택자. 무엇을 꾸밀 지 선택하고 속성을 넣어 예쁘게 꾸며주는 것.
```HTML
/* id selector */
#id{...}

/* class selector */
.class{...}

/* tag selector */
tagename{...}

/* 여러 요소 선택하기 */
#id, .class{...}

/* sudo selector */
/* 어떤 요소가 특정 상태(마우스 올림, 포커스 됨 등등)일 때만 선택하게 해주는 선택자 */
button:hover{...}

```
